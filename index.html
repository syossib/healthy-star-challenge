🔐 הגדרות סנכרון

כדי שכל המשפחה תראה את אותם הנתונים מכל מכשיר, צריך GitHub Token (פעם אחת לכל מכשיר):

  1. היכנסו ל-github.com/settings/tokens
  2. לחצו "Generate new token"
  3. תנו שם (למשל "healthy star"), בחרו תפוגה של 90 יום
  4. תחת Repository access בחרו "Only select repositories" ובחרו את healthy-star-challenge
  5. תחת Permissions ← Repository permissions ← Contents: בחרו "Read and write"
  6. לחצו "Generate token" והעתיקו
