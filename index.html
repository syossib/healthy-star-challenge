📖 הוראות הגדרה - סנכרון עם Google Sheets

הגדרה חד-פעמית (5 דקות) כדי שכל המשפחה תוכל לעקוב מכל מכשיר:

  1. צרו Google Sheet חדש
    היכנסו ל-sheets.google.com וצרו גיליון חדש. תנו לו שם, למשל "אתגר הכוכב הבריא".
  2. פתחו את עורך הסקריפט
    בגיליון, לחצו על תוספים (Extensions) ← Apps Script.
  3. מחקו את כל הקוד שמופיע והדביקו במקומו את הקוד הבא:
    function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Data'); if (!sheet) { return ContentService.createTextOutput('{"weeks":[]}') .setMimeType(ContentService.MimeType.JSON); } var val = sheet.getRange('A1').getValue(); return ContentService.createTextOutput(val || '{"weeks":[]}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Data'); if (!sheet) { sheet = SpreadsheetApp.getActiveSpreadsheet().insertSheet('Data'); } sheet.getRange('A1').setValue(e.postData.contents); sheet.getRange('A2').setValue(new Date().toISOString()); return ContentService.createTextOutput( JSON.stringify({status: 'ok'}) ).setMimeType(ContentService.MimeType.JSON); }
  4. שמרו את הסקריפט
    לחצו Ctrl+S (או Cmd+S ב-Mac) כדי לשמור.
  5. פרסו את הסקריפט כ-Web App
    לחצו על Deploy (פריסה) ← New deployment (פריסה חדשה).
    ליד "Type" לחצו על גלגל השיניים ובחרו Web app.
    בשדה "Who has access" בחרו: Anyone.
    לחצו Deploy.
  6. אשרו את ההרשאות
    Google יבקש אישור. לחצו Authorize access, בחרו את החשבון שלכם.
    אם מופיעה אזהרה "Google hasn't verified this app" - לחצו Advanced ← Go to... (זה בטוח - זה הסקריפט שלכם).
  7. העתיקו את כתובת ה-URL
    לאחר הפריסה, תופיע כתובת URL שמתחילה ב-https://script.google.com/macros/s/...
    העתיקו אותה והדביקו אותה בשדה "כתובת הסנכרון" למעלה.
  8. שתפו את ה-URL עם כל המשפחה!
    כל מי שמדביק את אותה כתובת באפליקציה יכול לסנכרן את אותם הנתונים.
💡 טיפ: אחרי ההגדרה, לחצו "שלח לענן" כדי לשמור את הנתונים. במכשיר אחר, הדביקו את אותה כתובת ולחצו "טען מהענן".
